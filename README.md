# Web-Design-Agency-Project-7

## Overview
This project is a responsive landing page for a creative design agency built with HTML and CSS.
It showcases a modern layout, clear typography, and a two-column card design that adapts to different screen sizes.
The main purpose of this project is to practice responsive design, layout techniques, and clean UI structure.

![image alt](https://github.com/Ferb-10/Web-Design-Agency-Project-7/blob/12e523c3a2ec2c635ade7ce53be8ea72a3620130/assets/images/web%20design%20agency.png)

## Features
- Agency-style landing page layout
- Two-column card design using floats
- Responsive design with media queries
- Custom typography using Google Fonts
- Image-based content sections

## Technologies Used
- HTML5
- CSS
- Google Fonts

## What I Learned
- Creating responsive layouts using media queries
- Building multi-column layouts with float
- Adapting layouts for mobile screens
- Using typography and color for branding
- Structuring reusable card components

## 概要
このプロジェクトは、HTMLとCSSを使用して作成した、
クリエイティブデザインエージェンシーのランディングページです。
2カラムレイアウトとレスポンシブ対応を取り入れ、画面サイズに応じて表示が変化する設計になっています。
実務を意識したレイアウト設計とUI構成の練習を目的としています。

## 機能
- ランディングページ
- floatを使った2カラムカードレイアウト
- メディアクエリによるレスポンシブ対応
- Google Fonts を使用したタイポグラフィ
- 画像を使ったコンテンツセクション

## 使用技術
- HTML5
- CSS
- Google Fonts

## 学んだこと
- メディアクエリを使ったレスポンシブ設計
- floatを用いた複数カラムレイアウト
- モバイル画面向けのレイアウト調整
- フォントと色を使ったブランディング表現
- 再利用しやすいカード構造の設計
